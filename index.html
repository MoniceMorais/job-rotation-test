Questão 4

Dois veículos (um carro e um caminhão) saem respectivamente de cidades opostas pela mesma rodovia. O carro de Ribeirão Preto em direção a Franca, a uma velocidade constante de 110 km/h e o caminhão de Franca em direção a Ribeirão Preto a uma velocidade constante de 80 km/h. Quando eles se cruzarem na rodovia, qual estará mais próximo a cidade de Ribeirão Preto?

Resposta: Quando os dois veículos se cruzarem na rodovia, eles terão percorrido juntos uma distância equivalente à distância entre Ribeirão Preto e Franca. Como ambos estão em velocidades constantes, podemos usar a fórmula de distância = velocidade x tempo para calcular a distância percorrida por cada veículo até o ponto de cruzamento.
Vamos chamar a distância entre as duas cidades de "d". Como o carro está viajando de Ribeirão Preto para Franca, ele percorrerá uma distância "d-x", onde "x" é a distância percorrida pelo caminhão em direção a Ribeirão Preto antes de se cruzar com o carro. O caminhão, por outro lado, percorrerá uma distância "x" em direção a Ribeirão Preto antes de se cruzar com o carro.
Podemos usar a fórmula de distância = velocidade x tempo para calcular o tempo que cada veículo levará para percorrer essas distâncias:
tempo do carro = (d-x) / 110
tempo do caminhão = x / 80

Igualando essas duas equações e resolvendo para x, obtemos:

(d-x) / 110 = x / 80
80(d-x) = 110x
80d - 80x = 110x
80d = 190x
x = (80/190)d
x = 0,421d

Portanto, quando os dois veículos se cruzarem, o caminhão estará a uma distância de 0,421d de Ribeirão Preto e o carro estará a uma distância de (1-0,421)d = 0,579d de Ribeirão Preto. Portanto, o carro estará mais próximo de Ribeirão Preto quando os dois veículos se cruzarem.
